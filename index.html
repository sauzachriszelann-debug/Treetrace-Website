About TreeTrace

Bridging Technology & Forest Conservation

TreeTrace was born from a critical need to replace fragmented, paper-based tree management with a unified digital system. It empowers LGUs, agroforestry organizations, and researchers with precision tools to monitor, identify, and protect Philippine forest resources.

🌐
SDG-Aligned Mission
Supports UN SDG 13 (Climate Action) and SDG 15 (Life on Land) through nature-based digital monitoring solutions.
🤖
AI-Driven at Its Core
Deep learning models process real-time tree images to identify species, estimate biomass, and classify conservation status automatically.
📡
Offline-First Design
Captures and stores data locally in areas without connectivity, then auto-syncs to the cloud — built for Philippine rural realities.
Core Features

Everything a Forest Needs to Thrive

Six integrated capabilities working in unison — from the field worker's camera to the LGU admin's dashboard — making tree management smarter, faster, and accessible.

01 — IDENTIFY
AI Species Identification

Point your camera at any tree. The semantic segmentation engine identifies the species in real time, displaying the common name, scientific name, and key information instantly.

02 — PROTECT
Conservation Status Alerts

Cross-references DENR DAO 2017-11 and the IUCN Red List to flag endangered or protected trees with do-not-cut alerts — safeguarding biodiversity in the field.

03 — MAP
Geo-Spatial GPS Mapping

Every tree is tagged with precision GPS coordinates (under 5m accuracy) and plotted on an interactive map — building a living digital twin of agroforestry sites.

04 — MEASURE
Biomass & Carbon Estimation

Automated allometric models calculate aboveground biomass and carbon sequestration potential from AI-estimated DBH — enabling carbon market participation for smallholders.

05 — SYNC
Offline-First Architecture

Record tree data deep in remote forests without internet. All entries are stored locally and automatically synchronized to the cloud the moment connectivity is detected.

06 — GROW
Crowdsourced AI Retraining

Unknown species? Submit a labeled photo. The community-driven retraining module continuously expands the AI model database — making TreeTrace smarter with every field encounter.

Process

From Forest Floor to Digital Record

A seamless four-step journey transforms a single tree encounter into verified, actionable conservation data — accessible by LGUs, researchers, and organizations across Davao del Norte.

01
Capture in the Field

Field workers use the mobile app to photograph a tree. GPS coordinates are auto-captured. Data is saved locally even without signal.

02
AI Analysis

The ML engine identifies the species via semantic segmentation, estimates DBH, classifies conservation status, and calculates carbon data.

03
Cloud Sync & QR Tag

Records sync automatically. A unique QR code is generated per tree — linking the physical tree to its complete digital profile.

04
Dashboard & Reports

Admins monitor species maps, community structure, carbon reports, and conservation alerts through the web dashboard in real time.

🧬
Semantic Segmentation (CNN)
Isolates tree trunk regions from photos for non-contact DBH measurement with 90%+ accuracy.
📏
Allometric Equations (Chave et al., 2014)
Tropical biomass models linking DBH, wood density, and height to verified aboveground carbon.
🗺️
GIS + Haversine Formula
Spherical-Earth distance calculations for precise tree-to-tree spatial mapping across agroforestry sites.
🔲
QR Code Integration
Each tree gets a permanent scannable identifier — enabling 70% faster data retrieval vs. manual lookup.
🌿
DENR DAO 2017-11 + IUCN Red List API
Real-time conservation status queries against Philippine and international endangered species databases.
☁️
FastAPI + SQL Cloud Backend
RESTful API serving mobile and web clients with real-time sync, role-based access control, and SMOTE-balanced retraining pipeline.
🏗️ System Architecture
TreeTrace uses a three-tier client-server architecture with a mobile field app, a web dashboard, and a cloud-hosted FastAPI + SQL backend. An offline-first local storage layer handles connectivity gaps in remote sites.

AI / ML PIPELINE (FastAPI)
Image → CNN Segment → DBH → Allometric → Carbon → DENR Check
Python FastAPI microservice with SMOTE-balanced training and IUCN Red List integration.
WEB DASHBOARD (ADMIN)
Species Map · Carbon Reports · Conservation Alerts · QR Generator
Real-time web dashboard with GIS map, community structure analysis, and report exports.
